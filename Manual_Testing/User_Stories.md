# User Stories

## 1. User Login
As a user, I want to securely log into the Cash Management Portal so I can access my financial dashboard.

## 2. View Account Balance
As a user, I want to see my account balances so I can monitor my financial activity.

## 3. Approve Payments
As a user, I want to approve pending vendor payments so that transactions can be completed.

## 4. Payment History
As a user, I want to view my past transactions so I can track historical payments.
